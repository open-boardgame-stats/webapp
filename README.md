This is a POC repo for OBGS. We'll start with a couple of pages with pre-generated data to see how things could look.

# Installation

Just run `yarn` and you are good to go.

## Workflow

Take an [unassigned issue](https://github.com/open-boardgame-stats/poc/issues?q=is%3Aopen+is%3Aissue+no%3Aassignee), assign it to yourself and work on it.

If you don't see an issue for what you'd like to work on - [create it](https://github.com/open-boardgame-stats/poc/issues/new).

Issues are used to synchronize work, to avoid having two people working on the same task.

Once you're done, create a PR and add `Closes #<issue number>` to the PR description to automatically close the issue when the PR is merged. If the issue is not complete - it's likely a large issue, and most likely worth splitting into smaller subtasks, with an issue for each. Split accoridngly, and close one of the newly split out issues in your PR.
